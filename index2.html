Para nosotros lo más importante en una aplicación es la forma con la que se muestra a los usuarios, por eso invertimos gran parte de nuestro tiempo en mejorar y crear nuevos diseños con los que obtener una mejor experiencia.

HTML5

es la última versión de HTML. El término representa dos conceptos diferentes: Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos. Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance

CSS

es un lenguaje utilizado en la presentación de documentos HTML. Un documento HTML viene siendo coloquialmente “una página web”. Entonces podemos decir que el lenguaje CSS sirve para organizar la presentación y aspecto de una página web.

JavasCript

es un lenguaje de programación, al igual que PHP, si bien tiene diferencias importantes con éste. JavaScript se utiliza principalmente del lado del cliente ,es decir, se ejecuta en nuestro ordenador, no en el servidor,permitiendo crear efectos atractivos y dinámicos en las páginas web.
